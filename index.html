A 股基本面分析平台

财报透析

基于 Python + AKShare 的 A 股基本面分析工具，四段式分析框架：业绩检验、业绩归因、验证排雷、估值区间。

  • 营收 / 净利润 / 毛利率 / ROE 趋势与 YoY 增速
  • 杜邦三因子分解与费用率分析
  • 利润含金量 / 收入质量 / 库存压力 排雷仪表盘
  • PE / PB 历史分位 + 三角验证估值区间
  • 主营业务构成与资产负债结构堆积图

本项目为本地运行的 Flask 应用，需要 Python 环境：

# 1. 克隆仓库
git clone https://github.com/Ju-chen77/stock-analyzer_X-C.git
cd stock-analyzer_X-C

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动
python app_V4.py

# 4. 打开浏览器访问
http://127.0.0.1:5000

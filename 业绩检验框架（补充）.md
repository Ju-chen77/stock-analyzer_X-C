# 业绩检验框架

> **核心目的**：拿到一份财报后，回答"业绩是真实的吗？盈利能力如何？数字之间是否自洽？"——这是三段式财务分析的第一道关卡。
>
> **定位**：三段式财务分析的入口。检验通过才进入归因；检验不通过直接进入排雷。
>
> **与其他模块的分工**：业绩检验回答"是什么"，业绩归因回答"为什么"，验证排雷回答"有没有雷"。

---

## 一、业绩检验的三层结构

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│   A · 盈利能力         B · 业绩自洽性        C · 现金流验证        │
│   ─────────────       ─────────────       ─────────────         │
│   业绩好不好？         数据自己打架吗？      利润换得来现金吗？     │
│                                                                  │
│   • 增长性             • 营收 vs 应收        • 三组含金量          │
│   • 盈利率             • 营收 vs 存货        • 自由现金流          │
│   • 回报率             • 营收 vs 合同负债    • 三类现金流组合      │
│   • 增长质量           • 毛利率 vs 同行                            │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
                              ↓
                  全部通过 → 进入业绩归因
                  自洽性或现金流不通过 → 直接进入验证排雷
```

**核心顺序**：A 看绝对水平（够不够好）→ B 看内部勾稽（自己是否打架）→ C 看真伪验证（账面 vs 现金）。任何一层亮红灯都要回头修正结论。

---

# A · 盈利能力指标

> **核心问题**：业绩自身的水平如何？增长是否健康？

---

## A-1：增长性指标

| 指标 | 计算 | 判读重点 |
|---|---|---|
| 营业收入 YoY | (本期营收 − 上年同期) / 上年同期 | 成长性的起点 |
| 营业收入 QoQ | (本期 − 上期) / 上期 | 短期动能、季节性 |
| 归母净利润 YoY | 同上 | 与营收增速对比看效率 |
| **扣非净利润 YoY** | 同上 | **真实盈利增速** |
| 营业利润 YoY | 同上 | 剔除税收影响的主业增速 |

### 增长质量的核心判读

| 情形 | 解读 |
|---|---|
| 营收 +20%，净利润 +25% | 健康增长（效率提升） |
| 营收 +20%，净利润 +60% | 警惕！可能靠非经常性损益拉动，需扣非验证 |
| 营收 +20%，扣非净利润 +5% | **非主业贡献**，主业增速实为 5% |
| 营收 +20%，QoQ 持续下滑 | 增速可能见顶 |
| 营收 −10%，净利润 +20% | 降本增效 or 一次性收益？需拆解 |

**核心原则**：永远用**扣非净利润**作为真实盈利增速，归母净利润看"账面表现"。

---

## A-2：盈利能力指标

| 指标 | 计算 | 反映 |
|---|---|---|
| 毛利率 | (营收 − 营业成本) / 营收 | 产品定价力 + 成本控制 |
| 营业利润率 | 营业利润 / 营收 | 主业盈利能力 |
| 净利率 | 归母净利润 / 营收 | 综合盈利能力 |
| **扣非净利率** | 扣非净利润 / 营收 | **剔除一次性损益的真实盈利能力** |

### 关键对比

| 对比组 | 含义 |
|---|---|
| 毛利率 − 净利率 | 费用率总和（销售/管理/研发/财务） |
| 净利率 − 扣非净利率 | 非经常性损益占比（> 2pct 需重点关注） |
| 毛利率 vs 历史均值 | 定价力变化趋势 |

---

## A-3：回报率指标

| 指标 | 计算 | 用途 |
|---|---|---|
| ROE | 归母净利润 / 平均归母净资产 | 综合回报水平 |
| ROA | 归母净利润 / 平均总资产 | 资产利用效率 |
| ROIC | EBIT × (1 − 税率) / (净资产 + 有息负债) | 排除杠杆影响的真实回报 |

> **ROE vs ROIC 的差异**：ROE 高但 ROIC 一般 → 主要靠加杠杆撑回报。详细归因见业绩归因框架的杜邦分解。

---

## A-4：趋势与对比判读

单期指标毫无意义，必须做三组对比：

| 对比维度 | 方法 | 判读 |
|---|---|---|
| **纵向对比** | 与自身 5–10 年历史均值/中位数对比 | 当前位置如何？是改善还是恶化？ |
| **横向对比** | 与同行业中位数对比 | 行业内排序如何？ |
| **行业基准** | 与申万一级/二级行业均值对比 | 是行业景气，还是个股 alpha？ |

# B · 业绩自洽性检验

> **核心逻辑**：报表内部各科目变化必须相互印证。营收暴涨但应收堆积、存货积压、毛利率离谱地高于同行——都是造假或经营恶化的早期信号。
>
> **核心原则**：**自洽性不通过 = 业绩可能有水分**，必须进入排雷阶段。

---

## B-1：营收 vs 应收账款

**判读逻辑**：营收增长应当主要换来现金或合理账期内的应收，应收增速持续高于营收增速 = 赊销冲量。

| 指标 | 计算 | 健康信号 | 警戒信号 |
|---|---|---|---|
| 应收账款增速 | YoY | ≤ 营收增速 | 超出营收增速 5pct+ |
| 应收账款 / 营业收入 | 同名 | 稳定或下降 | 持续上升 |
| 应收账款周转天数 | 365 / (营收 / 平均应收) | 稳定 | 持续拉长 |

### 典型情形

| 现象 | 暗示 |
|---|---|
| 营收 +30%，应收 +80% | 赊销冲量，回款堪忧 |
| 营收稳定，应收激增 | 收入质量恶化 |
| 应收 +50%，账龄结构恶化 | 坏账风险积累 |

---

## B-2：营收 vs 存货

**判读逻辑**：存货应当与销售匹配。存货大幅领先营收 = 滞销 / 行业景气下行 / 提前备货（需识别）。

| 指标 | 计算 | 健康信号 | 警戒信号 |
|---|---|---|---|
| 存货增速 | YoY | ≤ 营收增速 | 大幅领先营收 |
| 存货 / 营业收入 | 同名 | 稳定 | 持续上升 |
| 存货周转天数 | 365 / (营业成本 / 平均存货) | 稳定 | 持续拉长 |

### 行业特殊性

| 行业 | 高存货判读 |
|---|---|
| 房地产 | 存货 = 待售房产，周转天数 > 1000 天常态 |
| 白酒 / 奢侈品 | 高存货 = 升值资产（基酒越陈越值钱） |
| 消费电子 / 服装 | 高存货 = 高跌价风险 |
| 大宗商品 | 看周期阶段（底部备货可能是机会） |

---

## B-3：营收 vs 合同负债

**判读逻辑**：合同负债（原"预收账款"）= 客户已预付但尚未确认收入的部分，是**未来订单的领先指标**。

| 指标 | 计算 | 健康信号 | 警戒信号 |
|---|---|---|---|
| 合同负债增速 | YoY | 与营收同步或领先 | 大幅放缓或下降 |
| 合同负债 / 营收 | 同名 | 稳定或上升 | 持续下降 |

### 典型情形

| 现象 | 暗示 |
|---|---|
| 营收 +25%，合同负债 −40% | 未来订单严重萎缩，营收增速难持续 |
| 营收持平，合同负债 +50% | 订单储备充足，下期营收有望加速 |
| 营收 +20%，合同负债 +30% | 健康增长，订单充裕 |

> **行业差异**：合同负债适用于"先收款后交付"模式（白酒、机械、软件订阅、地产预售）；零售、To C 即时交易的行业意义不大。

---

## B-4：毛利率 vs 同行中位数

**判读逻辑**：毛利率显著高于同行而无合理解释 → 疑似虚增收入或低估成本。

| 偏离情形 | 可能原因 | 处理 |
|---|---|---|
| 高出同行 5pct 内 | 正常公司差异 | 接受 |
| 高出同行 5–15pct | 需明确解释（品牌溢价、技术壁垒、规模效应） | 重点核查 |
| 高出同行 15pct+ | 警惕造假，或商业模式独特（需多维验证） | 强烈警戒 |
| 低于同行中位数 | 竞争力弱、价格战、原材料劣势 | 关注变化趋势 |

### 同步核查项

毛利率高于同行时，必须同步检查：
- 现金流是否同步好转（含金量验证）
- 应收账款是否异常堆积（赊销虚增）
- 关联交易占比是否过高（关联方利益输送）
- 行业地位是否真的能支撑（产能、市占率、专利）

---

## B-5：自洽性失败的典型案例库

| 案例模式 | 报表特征 | 暗示 |
|---|---|---|
| **赊销冲量** | 营收暴增 + 应收激增 + 现金流恶化 | 收入质量低 |
| **压货式增长** | 营收增长 + 存货激增 + 经销商资金压力 | 透支未来 |
| **关联方虚增** | 高毛利 + 关联交易占比高 + 应收集中关联方 | 涉嫌财务造假 |
| **资本化造利润** | 净利率提升 + 开发支出激增 + 经营现金流恶化 | 利润纸面化 |

**图表**：✅ 四组双线对比图（5 年趋势）+ 行业毛利率分位数横向对比

# C · 现金流验证

> **核心逻辑**：利润表看"应计"（赚到 = 开了发票），现金流量表看"实收实付"（赚到 = 收到钱）。两者持续背离 = 业绩有水分。
>
> **定位**：业绩检验的最后一道防线，也是排雷的入口。**自洽性可能造假，但现金很难造假**——这是为什么现金流验证最关键。

---

## C-1：三组含金量对照

| 对照组 | 利润表科目 | 现金流量表科目 | 健康信号 |
|---|---|---|---|
| **收入含金量** | 营业收入 | 销售商品、提供劳务收到的现金 | 现金流入 / 营收 ≈ 0.95–1.17（含增值税） |
| **成本含金量** | 营业成本 | 购买商品、接受劳务支付的现金 | 比值稳定，无大幅背离 |
| **利润含金量** | 净利润 | 经营活动现金流量净额 | 经营现金流 / 净利润 ≥ 0.8 |

### 详细判读规则

**收入含金量**

| 比值 | 解读 |
|---|---|
| > 1.17 | 异常偏高（一般是预收增加） |
| 1.0–1.17 | 健康（含 13%/17% 增值税） |
| 0.9–1.0 | 略低，应收账款扩大 |
| < 0.9 持续 | **不健康**，赊销严重 |

**成本含金量**

| 现象 | 解读 |
|---|---|
| 现金流出 ≈ 营业成本 | 正常匹配 |
| 现金流出 >> 营业成本 | 大量备货（看景气阶段） |
| 现金流出 << 营业成本 | 应付账款扩大（占用上游资金，强势地位） |

**利润含金量（最关键）**

| 比值 | 解读 |
|---|---|
| ≥ 1.0 | 优质（如茅台、白酒龙头长期 > 1） |
| 0.8–1.0 | 健康 |
| 0.5–0.8 | 一般，需关注趋势 |
| < 0.5 持续 | **利润纸面化**，重点排雷信号 |
| 净利润 > 0 但经营现金流 < 0 | **强烈警示**，盈利质量极差 |

**图表**：✅ 三组双柱对比图（5 年时间序列）+ 比值折线叠加

```
┌─ 收入 vs 销售收到现金 ─┐  ┌─ 成本 vs 购买支付现金 ─┐  ┌─ 净利润 vs 经营现金流 ─┐
│   柱+比值线             │  │   柱+比值线             │  │   柱+比值线             │
└─────────────────────────┘  └─────────────────────────┘  └─────────────────────────┘
```

## C-2：自由现金流（FCF）检验

**逻辑**：经营现金流再多，如果都要砸进资本开支才能维持，对股东也无意义。FCF 才是真正"可分配给股东"的现金。

### 计算方式

| 口径 | 公式 | 适用 |
|---|---|---|
| **简化口径** | 经营现金流 − 资本开支（购建固定资产支付的现金） | 快速估算 |
| **精确口径** | 经营现金流 − 维持性资本开支（不含扩张性投资） | 价值评估，需附注拆解 |

### 判读

| FCF 情形 | 解读 |
|---|---|
| FCF 长期为正且稳定 | 优质，可持续分红/回购 |
| FCF 周期性波动（重资产） | 看完整周期是否净流入 |
| FCF 连续 3 年为负 | 增长靠烧钱，**不可持续** |
| FCF 为正但呈下降趋势 | 资本开支扩大，关注 ROIC 是否同步提升 |

> **行业差异**：钢铁、化工、半导体等重资产行业 FCF 在扩张期为负是正常的，需结合 ROIC 判断扩张是否值得。

---

## C-3：三类现金流组合判读

经营 / 投资 / 筹资 三类现金流的正负组合，反映公司所处阶段：

| 经营 | 投资 | 筹资 | 公司阶段 | 风险等级 |
|---|---|---|---|---|
| + | − | − | **健康成熟期**：自我造血 + 扩张 + 还债/分红 | 低 |
| + | − | + | **成长扩张期**：自我造血 + 大举投资 + 借钱补充 | 中 |
| + | + | − | **收缩调整期**：经营+卖资产还债 | 中 |
| + | + | + | 现金积累期（少见） | 低 |
| − | − | + | ⚠️ **靠融资生存** | 高 |
| − | + | + | ⚠️⚠️ **变卖资产 + 借钱** | 极高 |
| − | + | − | 收缩自救期 | 高 |
| − | − | − | 三流皆出（崩溃前夜） | 极高 |

### 多年组合判读

| 情形 | 解读 |
|---|---|
| 连续 3 年 (+, −, −) | 优质成熟公司（茅台、长江电力） |
| 连续 3 年 (+, −, +) | 健康成长公司（中等规模制造业） |
| 任何一年含 (−, +, +) | 严重警示，必须深入排雷 |

**图表**：✅ 现金流三项流向桑基图 / 三色柱状图（5 年时间序列）

# 完整检验流程

```
1. 拉取数据
   ↓ akshare 三大报表 + 行业可比公司
2. A · 盈利能力扫描
   ↓ 增长性 + 盈利率 + 回报率 + 趋势对比
   ├─ 通过 → 继续
   └─ 不通过 → 业绩较差，看归因找原因
3. B · 业绩自洽性检验
   ↓ 4 组对照
   ├─ 通过 → 继续
   └─ 不通过 → 业绩可能有水分，进入 C
4. C · 现金流验证
   ↓ 三组含金量 + FCF + 三类组合
   ├─ 通过 → 进入业绩归因
   ├─ 含金量不通过 → 直接进入排雷
   └─ FCF / 现金流组合警示 → 直接进入排雷
5. 输出业绩检验报告
   ↓
   ├─ verdict: PASS / WARNING / FAIL
   └─ 后续路径：归因 or 排雷
```

# financials.py

# ─── B · 业绩自洽性 ───

# ─── 主入口 ───

# 关键原则

1. **检验先于归因**：先确认业绩是真实的，再分析为什么——倒做会得出错误结论
2. **三层独立判断**：A 好不代表 B 好，B 好不代表 C 好——任何一层不通过都要追究
3. **扣非优先**：所有增速、利润率指标优先看扣非口径，避免一次性损益干扰
4. **趋势 > 单期**：每个指标看 5 年趋势，单期数据不算数
5. **三组对比同时做**：自身历史 + 行业中位数 + 行业景气，三者缺一不可
6. **现金为王**：自洽性可能造假，含金量难造假——现金流验证是底线
7. **业绩太完美要警惕**：所有指标都"恰到好处"反而异常，业绩平稳的造假比业绩波动的造假更难识别

---

**一句话总结**：业绩检验的本质是**"三重核对"**——账面增长是否合理（盈利能力）、报表内部是否打架（自洽性）、利润是否换得来真金白银（现金流）。三关都过，业绩才算"真的好"。

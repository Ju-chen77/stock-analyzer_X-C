# 盈利预测框架

> **核心目的**：基于历史财务、行业景气、产品结构与一致预期，对企业未来 1–3 年的**收入、毛利率、费用率、净利润、EPS** 给出**三档情景预测**，作为估值的输入。
>
> **定位**：业绩检验和业绩归因之后、PE/PB 估值之前的**关键前置环节**。估值的核心不确定性 80% 来自预测假设而非估值倍数。
>
> **核心原则**：永远输出**区间和情景**，禁止单点预测；所有假设必须**显式化**，便于复盘。

---

## 落地说明（Tab ⑤ 盈利预测 · 实际实现 vs 框架）

> 本框架已落地为 **Tab ⑤ 盈利预测**（后端 `compute_forecast()`，数据复用主分析的历史利润表，无额外网络请求）。
> 受本地代理环境数据可达性限制，对框架做了如下**可行性调整**：

| 框架模块 | 实际落地 | 说明 |
|---|---|---|
| 方法 A · 历史 CAGR 外推 | ✅ 已实现 | 3 年 / 5 年 CAGR 加权（取均值），防极端值裁剪到 [−50%, +100%] |
| 方法 B · 一致预期取数/调整 | ❌ 未实现 | `stock_profit_forecast_em` 在本地代理环境不可用（返回异常），无法取卖方一致预期 |
| 方法 C · 量价模型（销量 × ASP）| ❌ 未实现 | 需分产品销量/ASP/行业规模，akshare 不提供，需 PDF/手动维护（框架亦列为 P3） |
| 毛利率预测 | ✅ 简化 | 采用「路径 1 直接外推」：基准 = 最近一年毛利率；情景 ±2pct/±1.5pct |
| 费用率预测 | ✅ 已实现 | 历史 3 年均值 − 规模效应摊薄（按营收增速分档 0/0.35/0.75/1.5pct）|
| 净利润推导 | ✅ 调整 | **以「当前实际归母净利率」为锚**，按毛利率/费用率相对基准的偏离（税后）调整净利率。<br>而非纯自下而上 营收×毛利率−费用−税——后者对投资收益/减值/少数股东损益占比大的控股型公司（如 TCL）会系统性失真 |
| 三档情景 | ✅ 已实现 | 悲观/基准/乐观：营收增速 ±10pct、毛利率 −2/+1.5pct、费用率 +0.5/−1pct |
| 敏感性矩阵 | ✅ 已实现 | 营收增速 × 毛利率 5×5 → 归母净利润热力图 |
| 单变量冲击 | ✅ 已实现 | 营收增速 +10pct / 毛利率 +2pct / 费用率 +1pct / 有效税率 +2pct → 归母净利润变化% |
| 3 年轨迹 | ✅ 已实现 | 基准情景复利外推 T+1/T+2/T+3 |
| 预期差量化（第七节）| ❌ 未实现 | 依赖方法 B 的一致预期，数据源不可用，暂缺；后续可在可直连环境补充 |

**净利率锚方法说明**：基准归母净利率 = 当前实际归母净利润 / 营收（缺失退 3 年均值）；
情景归母净利率 = 锚 + [(情景毛利率 − 基准毛利率) − (情景费用率 − 基准费用率)] ×(1 − 有效税率)。
这样基准情景能复现近年实际盈利水平，情景再据毛利率/费用率假设的边际变化调整，兼顾「可复盘」与「与历史口径一致」。

## 一、盈利预测的三档情景

```
┌──────────────────────────────────────────────────────────┐
│                                                           │
│   悲观情景         基准情景         乐观情景               │
│   ─────────       ─────────        ─────────             │
│   下行风险         合理预期          上行空间               │
│                                                           │
│   • 行业景气下行    • 行业平稳         • 超预期景气          │
│   • 市占率下滑      • 市占率稳定       • 市占率提升          │
│   • 毛利率承压      • 毛利率稳定       • 毛利率扩张          │
│   • 一致预期 × 0.8  • 一致预期         • 一致预期 × 1.2     │
│                                                           │
└──────────────────────────────────────────────────────────┘
                          ↓
              输出三套 EPS → 喂入估值模型
                          ↓
              估值区间 [V_low, V_mid, V_high]
```

| 情景 | 输出 | 用途 |
|---|---|---|
| 悲观 | 下行 EPS | 用于估值下沿 / 安全边际判断 |
| 基准 | 中性 EPS | 用于合理估值锚 |
| 乐观 | 上行 EPS | 用于估值上沿 / 上行空间测算 |

**关键原则**：不要预测"目标价"，而是**预测三组数字喂给估值模型，让模型输出区间**。

# 二、收入预测：三种方法

收入预测决定整个盈利预测的方向，**三种方法应交叉验证而非二选一**。

---

## 方法 A · 历史增速外推（CAGR）

**适用**：业务稳定、行业增速可预测的成熟企业。

### 公式

```
未来收入 = 当期收入 × (1 + 加权 CAGR)^N
```

### 加权 CAGR 选取

| 加权方式 | 适用场景 |
|---|---|
| 近 3 年 CAGR | 业务持续向上 |
| 近 5 年 CAGR | 业务穿越周期 |
| 近 3 年与近 5 年的均值 | 综合考虑短中期 |
| 剔除异常年份的 CAGR | 有疫情 / 一次性影响 |

### 优势与陷阱

| 优势 | 陷阱 |
|---|---|
| 简单、客观、可复现 | 假设"过去 = 未来"，对拐点完全无感 |
| 对周期股能识别底部 | 对高速增长股严重低估 |
| 适合横向对比 | 对新业务、新产品不适用 |

> ⚠️ **致命陷阱**：成长股用此法会**系统性低估**。例：2023 年用 CAGR 外推中际旭创 2026 年营收，会得到约 200 亿（实际预期 780 亿）。

---

## 方法 B · 一致预期取数 / 调整

**适用**：有较好卖方覆盖的中大盘股；作为基准情景的锚点。

### 数据获取

| 数据 | 来源 |
|---|---|
| 一致预期净利润（2026E/2027E/2028E）| `ak.stock_profit_forecast_em()` |
| 覆盖分析师数量 | 同上 |
| 一致目标价 | `ak.stock_research_report_em()` |
| 预测发布日期 | 同上 |

### 调整规则

| 情形 | 调整方向 |
|---|---|
| 一致预期覆盖 < 3 家分析师 | 不采信，回退到方法 A |
| 一致预期 vs 历史趋势严重偏离 | 重新审视，可能某些黑天鹅 |
| 一致预期发布超过 3 个月 | 用最新季报修正 |
| 公司业绩持续超预期 | 一致预期 × 1.1–1.2（卖方滞后） |
| 公司业绩持续低于预期 | 一致预期 × 0.8–0.9 |

### akshare 短板

| 问题 | 应对 |
|---|---|
| 小盘股、ST 股可能无覆盖 | 回退到方法 A |
| 覆盖数据滞后 | 优先选取近 60 天内的研报 |
| 不同卖方分歧大 | 用中位数而非均值 |

> **核心用法**：一致预期取数 = 基准情景；× 0.8 = 悲观；× 1.2 = 乐观。

---

## 方法 C · 量价模型（销量 × ASP）

**适用**：制造业、消费、资源等可拆解出"量 × 价"的行业。

### 拆解公式

```
未来收入 = Σ(各产品销量 × 各产品 ASP)
        = Σ(行业规模 × 公司市占率 × 公司价格相对水平)
```

### 三级拆解

| 层级 | 拆解 | 数据来源 |
|---|---|---|
| **L1：分产品** | 营收按产品线拆 | 年报附注 |
| **L2：量价分离** | 销量 + ASP | 年报附注 / 路演 PPT |
| **L3：行业 × 市占率** | 行业规模 × 公司市占率 | 第三方研究报告 |

### 量价预测的内在逻辑

| 变量 | 影响因子 |
|---|---|
| **销量** | 行业景气、产能、市占率、客户绑定 |
| **ASP** | 产品迭代周期（推出 → 良率爬坡 → 价格趋稳 → 旧品萎缩） |
| **新品 vs 旧品** | 新品溢价 ~50%（窗口期），旧品年降 15–25% |

### ASP 周期三阶段

```
推出初期         良率爬坡期           成熟期
─────────       ─────────────        ─────────
新品溢价高       价格快速下降          年降趋稳
ASP × 1.0       ASP × 0.5-0.7        ASP × 0.3-0.5
12 个月          12-24 个月            > 24 个月
```

> **示例**（中际旭创报告）：
> - 800G ASP: 推出初 $1,100 → 当前 $380（累计 -65%）
> - 1.6T ASP: 推出初 $1,400 → 当前 $1,100（累计 -21%，仍在窗口期）
> - 新品溢价不持久，**唯一可控变量是"新品收入占总收入的比例"**

### akshare 数据限制

| 数据 | 可得性 | 替代方案 |
|---|---|---|
| 分大类产品收入 | ✅ 年报附注（pdfplumber 解析） | — |
| 分产品销量 | ⚠️ 部分公司披露总销量，分类拆分罕见 | 路演 PPT / 电话会纪要手动录入 |
| ASP | ⚠️ 大部分需自行计算（收入 / 销量）| 同上 |
| 行业规模 | ❌ akshare 不提供 | LightCounting / TrendForce（付费）或 Wind |
| 市占率 | ❌ 同上 | 同上 |

> **现实建议**：方法 C 在 MVP 阶段难落地，先用方法 A + B 跑通，方法 C 作为高价值标的的**深度研究模块**手动维护。

---

## 三种方法的交叉验证

| 三方法结论 | 判读 |
|---|---|
| 三者一致 | 高置信，作为基准 |
| A < B < C | 业务加速期（量价模型最准） |
| A > B > C | 业务减速期（一致预期可能滞后） |
| B 明显偏离 A、C | 卖方预期可能过度乐观 / 悲观 |
| 三者分歧 > 30% | 不确定性极大，扩大情景区间 |

# 三、毛利率预测

毛利率是盈利预测中**第二大不确定来源**（仅次于收入），但**远比收入更难预测**。

---

## 三层预测逻辑

```
┌──────────────────────────────────────────────────┐
│  L1：产品结构变化                                  │
│      新品占比 ↑ → 整体毛利率 ↑（窗口期）           │
├──────────────────────────────────────────────────┤
│  L2：单产品生命周期                                │
│      新品溢价 → 良率爬坡降本 → 成熟期年降          │
├──────────────────────────────────────────────────┤
│  L3：行业景气与成本                                │
│      原材料价格、产能利用率、价格战程度             │
└──────────────────────────────────────────────────┘
```

---

## L1 · 产品结构变化

**核心公式**：

```
整体毛利率 = Σ(各产品收入占比 × 各产品毛利率)
```

| 情形 | 毛利率方向 |
|---|---|
| 高毛利新品占比 ↑ | 整体毛利率 ↑ |
| 旧品占比 ↑ | 整体毛利率 ↓ |
| 产品结构稳定 | 看 L2、L3 |

> **示例**（中际旭创）：2024 → 2025 毛利率从 33.8% → 42.0%（+8.2pct），核心驱动是 800G/1.6T 占比快速提升。

---

## L2 · 单产品生命周期

| 阶段 | 毛利率特征 | 关键变量 |
|---|---|---|
| 推出初期（0–12 个月）| 高位，新品溢价 50%+ | 良率（50–60%）|
| 良率爬坡期（12–24 个月）| 快速下行 | 良率（60–80%）、竞争涌入 |
| 成熟期（> 24 个月）| 年降趋稳，15–25% | 行业竞争、原材料 |
| 衰退期 | 持续下行至接近成本线 | 替代品挤压 |

---

## L3 · 行业景气与成本

| 变量 | 数据来源 | 判读 |
|---|---|---|
| 上游原材料价格 | 商品期货、PMI 分项 | 上行 → 毛利率承压 |
| 行业产能利用率 | 行业研究报告、统计局 | 高位 → 提价能力 |
| 同行价格战程度 | 行业新闻、龙头价格 | 加剧 → 毛利率下行 |

---

## 毛利率预测的实操方法

| 路径 | 适用 |
|---|---|
| **路径 1：直接外推** | 业务稳定，毛利率 5 年波动 < 3pct |
| **路径 2：均值回归** | 周期性行业，预测向 5–10 年均值回归 |
| **路径 3：产品结构推导** | 有清晰的新旧产品迭代节奏 |
| **路径 4：同行对标** | 用行业龙头毛利率作天花板 |

### 三档假设示例

| 情景 | 毛利率假设 |
|---|---|
| 悲观 | 当期毛利率 − 2pct（价格战、原材料压力）|
| 基准 | 当期毛利率 ± 0.5pct |
| 乐观 | 当期毛利率 + 1–2pct（产品升级、规模效应）|

---

# 四、费用率预测

费用率比毛利率**更可预测**——大公司期间费用率通常稳定在窄区间。

---

## 四项费用率的预测逻辑

| 费用率 | 预测逻辑 | 趋势 |
|---|---|---|
| **销售费用率** | 渠道结构稳定 → 与营收基本同步 | 规模扩张时**温和下降** |
| **管理费用率** | 组织扩张速度 < 营收增速 → 规模摊薄 | 通常**逐年下降** |
| **研发费用率** | 战略投入，与营收增速可能不匹配 | 高成长股**稳定或上升** |
| **财务费用率** | 取决于资本结构（有息负债 / 货币资金）| 看融资活动 |

---

## 期间费用率预测公式

```
未来期间费用率 = Max(
    历史 3 年均值 - 规模效应摊薄(0–1pct),
    行业中位数下限
)
```

### 规模效应摊薄程度

| 营收增速 | 期间费用率摊薄幅度 |
|---|---|
| < 10% | 几乎无摊薄 |
| 10–30% | 摊薄 0.2–0.5pct |
| 30–60% | 摊薄 0.5–1pct |
| > 60% | 摊薄 1–2pct |

---

## 三档假设示例

| 情景 | 期间费用率假设 |
|---|---|
| 悲观 | 历史均值 + 0.5pct（投入扩张但收入不及预期）|
| 基准 | 历史均值（规模效应抵消扩张投入）|
| 乐观 | 历史均值 − 1pct（规模效应显著）|

---

# 五、净利润推导 + EPS 计算

---

## 推导路径

```
营业收入（预测）
 × 毛利率（预测）
─────────────
营业毛利
 − 营业收入 × 期间费用率（预测）
 − 营业收入 × 税金及附加率（取历史均值）
─────────────
营业利润
 + 投资收益 / 公允价值变动（保守假设为 0 或历史均值）
 − 所得税（按实际有效税率，取历史 3 年均值）
─────────────
净利润
 × 归母比例（取历史均值）
─────────────
归母净利润
 ÷ 总股本（考虑增发 / 回购）
─────────────
EPS
```

---

## 关键假设备注

| 项目 | 假设方式 |
|---|---|
| 税金及附加率 | 取历史 3 年均值（通常 0.5–2%）|
| 投资收益 / 公允价值 | 保守预测为 0，激进可取历史均值 |
| 所得税率 | 取实际有效税率（≠ 名义税率）|
| 归母比例 | 取历史 3 年均值（看是否有少数股东）|
| 股本变动 | 考虑未来 12 个月已公告的增发 / 回购 |

# 六、敏感性分析

> 一份盈利预测如果没有敏感性分析，**就不是预测，是赌博**。

---

## 敏感性矩阵：营收增速 × 毛利率

以净利润为输出变量，构建 5×5 敏感性矩阵：

| | 毛利率 −2pct | 毛利率 −1pct | **毛利率基准** | 毛利率 +1pct | 毛利率 +2pct |
|---|---|---|---|---|---|
| 营收增速 −20% | 极悲观 | | | | |
| 营收增速 −10% | | 悲观 | | | |
| **营收增速基准** | | | **基准** | | |
| 营收增速 +10% | | | | 乐观 | |
| 营收增速 +20% | | | | | 极乐观 |

**输出**：每个组合对应一个净利润数字，看分布和极端值。

---

## 单变量弹性

| 变量 | ±10% 变化导致的净利润变化 |
|---|---|
| 营收 | 弹性约 1.5–2.5x（含费用杠杆）|
| 毛利率 | 弹性约 2.5–4x（直接影响利润）|
| 期间费用率 | 弹性约 1.5–2x |
| 所得税率 | 弹性约 0.5x |

> **结论**：毛利率假设错 2pct 比营收预测错 10% 影响更大。**预测的精力应优先放在毛利率上**。

# 七、预期差量化

> **预期差 = 我们的预测 − 市场一致预期**
>
> 预期差是估值修复的核心驱动力。一致预期已经被 price in，唯有**自建预测与一致预期的差**才能带来超额收益。

---

## 预期差对比表

| 维度 | 市场一致预期 | 我们的判断 | 差异 | 上行/下行空间 |
|---|---|---|---|---|
| 营收增速 | a% | b% | b − a | 影响估值倍数 |
| 毛利率 | c% | d% | d − c | 影响净利润弹性 |
| 净利润 | e 亿 | f 亿 | (f − e) / e | 直接换算估值差 |
| 增长持续性 | n 年 | m 年 | m − n | 影响远期估值 |

---

## 预期差的四大常见来源

| 来源 | 描述 | 验证方法 |
|---|---|---|
| **市占率被低估** | 龙头优势未被充分定价 | 跟踪行业数据、龙头招标份额 |
| **毛利率韧性被低估** | 新品溢价 + 成本下降被忽视 | 拆解硅光 / 国产替代等降本路径 |
| **产能与订单不匹配被误读** | 在建工程被解读为过剩 | 验证订单能见度、上游瓶颈 |
| **高 PE 被认为透支** | TTM PE 高但前瞻 PE 合理 | 用前瞻 PE 替代 TTM |

## 预期差 → 估值修正

| 自建 vs 一致 | 含义 | 估值含义 |
|---|---|---|
| 自建 > 一致 30%+ | 强超预期 | 估值有显著上行空间 |
| 自建 > 一致 10–30% | 温和超预期 | 修正空间存在 |
| 自建 ≈ 一致 ±10% | 与共识一致 | 估值已合理 |
| 自建 < 一致 10–30% | 低于共识 | 警惕下修 |
| 自建 < 一致 30%+ | 强烈低于共识 | 估值有显著下行风险 |

> **核心原则**：当自建预测显著偏离一致预期时，**必须给出明确的偏离理由和证据链**——没有理由的偏离 = 拍脑袋。

# 八、完整预测流程

```
1. 数据准备
   ↓ 业绩检验完成（确认历史数据真实）
   ↓ 业绩归因完成（理解增长驱动因子）
2. 收入预测
   ↓ 方法 A: 历史 CAGR 外推
   ↓ 方法 B: 一致预期取数 ± 调整
   ↓ 方法 C: 量价模型（如有数据）
   ↓ 三方法交叉验证 → 基准营收
3. 毛利率预测
   ↓ L1 产品结构 + L2 生命周期 + L3 行业景气
   ↓ 基准毛利率
4. 费用率预测
   ↓ 历史均值 ± 规模效应
   ↓ 基准期间费用率
5. 净利润推导
   ↓ 营收 × 毛利率 − 费用 − 税
   ↓ 基准净利润 → 基准 EPS
6. 三档情景生成
   ↓ 悲观 / 基准 / 乐观 三套假设
   ↓ 三套 EPS
7. 敏感性分析
   ↓ 营收 × 毛利率 矩阵
   ↓ 单变量弹性
8. 预期差量化
   ↓ vs 一致预期
   ↓ 修正空间评估
9. 输出预测报告
   ↓ EPS 区间 → 喂入 PE_PB 估值模型
```

# 十一、关键原则

1. **永远三档情景，禁止单点预测**：估值的不确定性主要来自盈利预测
2. **毛利率比收入更重要**：弹性大 2 倍，预测精力优先放在毛利率
3. **三方法交叉验证**：单一方法都不可靠，至少两种方法佐证
4. **一致预期是锚不是真理**：卖方系统性滞后，对成长股偏低、对衰退股偏高
5. **假设必须显式化**：每一个数字背后必须有可追溯的假设和理由
6. **敏感性优先于精确度**：与其纠结小数点，不如把范围算清楚
7. **预期差才有 alpha**：与一致预期一样 = 没价值，要找差异
8. **量价模型最准但最难**：能做就做，做不了用方法 A+B 也能用
9. **基准情景 ≠ 中间值**：基准是"最可能发生"，不一定在悲观/乐观正中间
10. **预测要有迭代节奏**：季报后必须更新，黑天鹅事件后必须重做

**一句话总结**：盈利预测的本质是**"把不确定性量化"**——不是猜一个数字，而是用三种方法、三档情景、敏感性矩阵和预期差对比，把"未来业绩"的可能区间画清楚。估值模型再精巧，输入错了一切归零；**预测准确性 80% 决定了估值有效性**。

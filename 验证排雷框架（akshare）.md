# 验证排雷框架

> **核心目的**：识别财报中的造假信号、资产隐患、治理风险与可持续性问题，**在估值之前先排除"价值陷阱"**。
>
> **定位**：三段式财务分析的最后一道防线。
>
> **与业绩检验的分工**：业绩检验已经覆盖报表内部勾稽（自洽性）和利润真伪（现金流），本框架聚焦**资产质量、偿债能力、治理风险**——这些是估值前的最后一道防线。

---

## 一、排雷的三大维度

```
┌──────────────────────────────────────────────────────┐
│                                                       │
│   A · 资产端       B · 偿债与杠杆     C · 治理与外部   │
│   ─────────       ─────────────      ─────────────   │
│   资产里有雷？     还得起钱吗？        人/制度有问题？  │
│                                                       │
│   • 商誉          • 资产负债率        • 大股东质押     │
│   • 应收账款       • 利息保障倍数      • 审计意见       │
│   • 存货          • 有息负债/现金     • 高管变动       │
│   • 资本化研发     • 短期偿债         • 关联交易       │
│   • 其他应收       • 经营现金流覆盖   • 监管问询        │
│                                                       │
└──────────────────────────────────────────────────────┘
                        ↓
              D · 排雷仪表盘（红黄绿）
                        ↓
              E · 行业差异化阈值校准
                        ↓
              一票否决判定
```

每一维度独立判断，**任何一项亮红灯都必须警惕**，不能用其他维度的绿灯抵消。

---

# A · 资产端风险

> **核心问题**：资产负债表"资产"侧的项目里，哪些可能是虚的、坏的、要减值的？

---

## A-1：商誉减值

并购溢价形成的商誉是 A 股财务最大雷区之一，2018 年商誉减值潮的教训犹在。

| 指标 | 计算 | 警戒阈值 |
|---|---|---|
| **商誉占比** | 商誉 / 净资产 | > 30% 高危，> 50% 极高危 |
| **商誉绝对额** | 商誉余额 | 持续高位且未充分减值 |
| **被并购方业绩承诺兑现** | 实际利润 / 承诺利润 | < 80% 减值风险大 |
| **商誉减值测试假设** | 折现率、永续增长率 | 假设过于乐观 → 后续暴雷 |

**典型雷区**：业绩承诺期满后第一年（"承诺到期年"）是商誉减值高发期。

---

## A-2：应收账款风险

| 指标 | 计算 | 警戒信号 |
|---|---|---|
| **应收占比** | 应收账款 / 营业收入 | 持续上升 |
| **应收周转天数** | 365 / (营收 / 平均应收) | 持续拉长 |
| **账龄结构** | 1 年以内 / 1-2 年 / 2-3 年 / 3 年以上占比 | 长账龄占比上升 |
| **应收前五大客户集中度** | 前五大应收 / 应收总额 | 高集中度 + 客户质量差 |
| **坏账计提比例** | 坏账准备 / 应收账款 | 计提比例显著低于同行 → 利润虚高 |

**附注关键信息**（akshare 不直接提供，需从年报 PDF 解析）：
- 账龄分布
- 单项计提的大额应收明细
- 关联方应收占比

---

## A-3：存货跌价风险

| 指标 | 计算 | 警戒信号 |
|---|---|---|
| **存货周转天数** | 365 / (营业成本 / 平均存货) | 持续上升 |
| **存货 / 营收** | 同名 | 行业景气下行时尤其警惕 |
| **存货跌价准备 / 存货** | 跌价准备占比 | 计提不足 → 暴雷风险 |
| **库龄结构** | 类似账龄分析 | 长库龄占比上升 |

**行业差异**：
- 房地产：存货周转 > 1000 天是常态，看销售去化率
- 白酒、奢侈品：高存货是优势（增值）
- 消费电子、服装：高存货 = 高跌价风险

---

## A-4：资本化研发

研发支出可"费用化"（直接计入当期损益）或"资本化"（计入开发支出，变成无形资产）。**过度资本化是虚增利润的常见手段**。

| 指标 | 计算 | 警戒信号 |
|---|---|---|
| **资本化比例** | 开发支出 / 研发总投入 | 突然大幅上升 |
| **同行对比** | 资本化比例 vs 行业中位数 | 显著高于同行 |
| **资本化政策稳定性** | 历年是否一致 | 政策变更需重点关注 |

**关键判读**：研发资本化本身合规，但**比例突变 + 业绩压力期** = 强烈嫌疑。

---

## A-5：其他应收款 / 长期股权投资

这两个科目是"资金占用"和"利益输送"的高发区。

| 指标 | 警戒信号 |
|---|---|
| **其他应收款 / 总资产** | 异常高（> 5%）且去向不明 |
| **其他应收款 前五大** | 关联方或不明主体占比高 |
| **长期股权投资 / 净资产** | 大额投资非主业相关公司 |
| **对外担保 / 净资产** | > 50% 高危 |

---

# B · 偿债与杠杆风险

> **核心问题**：公司还得起钱吗？流动性紧张吗？是否靠借新还旧维持？

---

## B-1：杠杆水平

| 指标 | 计算 | 警戒阈值（参考） |
|---|---|---|
| 资产负债率 | 总负债 / 总资产 | **行业差异极大**，需对比同行 |
| 有息负债率 | (短期借款 + 一年内到期非流动负债 + 长期借款 + 应付债券) / 总资产 | 比资产负债率更有意义 |
| 净负债率 | (有息负债 − 货币资金) / 净资产 | > 100% 偏高 |

---

## B-2：短期偿债能力

| 指标 | 计算 | 警戒阈值 |
|---|---|---|
| 流动比率 | 流动资产 / 流动负债 | < 1 流动性紧张 |
| 速动比率 | (流动资产 − 存货) / 流动负债 | < 1 高危 |
| **有息负债 / 货币资金** | 同名 | **> 2 高危** |
| 短期借款 / 货币资金 | 同名 | > 1 流动性紧张 |

> 🚨 **重点警示模式**：**货币资金充裕 + 大额有息负债同时存在**
>
> 这是典型造假信号（康得新、康美药业模式）——账上有几十亿现金却还要高息借款，要么钱是假的（虚增），要么钱是真的但用不了（被冻结或被大股东占用）。

---

## B-3：偿债的现金流覆盖

利息和本金最终要用真金白银还，不能只看账面比率。

| 指标 | 计算 | 警戒阈值 |
|---|---|---|
| 利息保障倍数 | EBIT / 利息费用 | < 3 偏弱，< 1 危险 |
| **现金流利息保障** | 经营现金流 / 利息费用 | < 1 严重 |
| 经营现金流 / 流动负债 | 同名 | 看偿债能力的现金流口径 |
| 经营现金流 / 有息负债总额 | 同名 | > 0.2 健康 |

---

# C · 治理与外部信号

> **核心问题**：人、制度、外部监管层面有没有问题？这些信号往往比报表数字更早暴露风险。

---

## C-1：股权与质押

| 信号 | 警示等级 |
|---|---|
| 大股东股权质押比例 > 50% | 高 |
| 大股东股权质押比例 > 70% | 极高 |
| 质押爆仓线临近股价 | 极高 |
| 控股股东累计减持 > 5% | 中–高 |
| 高管集中减持 | 高 |
| 大股东占用上市公司资金 | **极高（强制退市风险）** |

---

## C-2：审计与信披

| 信号 | 来源 | 警示等级 |
|---|---|---|
| 审计意见为"非标准无保留" | 年报首页 | **极高** |
| 带强调事项段的无保留意见 | 年报首页 | 高 |
| 审计机构频繁更换 | 公告 | 高 |
| 财务总监 / 董秘频繁更换 | 公告 | 高 |
| 年报披露推迟 | 公告 | 高 |
| 业绩快报与正式年报差异显著 | 对比公告 | **极高** |
| 业绩预告多次修正 | 公告 | 中–高 |

> **审计意见解读**：从轻到重为「标准无保留」→「带强调事项段无保留」→「保留意见」→「无法表示意见」→「否定意见」。后三种统称"非标"。

---

## C-3：关联交易与担保

| 指标 | 警戒信号 |
|---|---|
| 关联交易 / 营业收入 | 占比 > 30% 且无业务必要 |
| 关联方应收占比 | 大额且持续增长 |
| 对外担保 / 净资产 | > 50% 高危 |
| 对关联方担保 | 任何金额都需警惕 |
| 关联方资金占用 | **任何金额，强制退市风险** |

---

## C-4：监管信号

| 信号 | 警示等级 |
|---|---|
| 交易所问询函 | 中（频繁则高） |
| 监管警示函 | 高 |
| 立案调查 | **极高** |
| 强制信披延期 | 极高 |
| 风险警示（ST / *ST） | 极高 |
| 退市预警 | 极高 |

# D · 排雷仪表盘（红黄绿三色）

将所有指标汇总成统一的状态表，便于快速决策。

## D-1：仪表盘标准

| 维度 | 指标 | 红线 🔴 | 黄线 🟡 | 绿线 🟢 |
|---|---|---|---|---|
| **资产** | 商誉 / 净资产 | > 50% | 30–50% | < 30% |
| 资产 | 应收 / 营收 趋势 | 连续上升 3 年 | 上升 2 年 | 稳定或下降 |
| 资产 | 存货周转天数同比 | +50% | +20% | 持平或下降 |
| 资产 | 资本化研发比例 | 突变 +30%+ | 突变 10–30% | 稳定 |
| 资产 | 其他应收 / 总资产 | > 10% | 5–10% | < 5% |
| **偿债** | 有息负债 / 货币资金 | > 2.0 | 1.0–2.0 | < 1.0 |
| 偿债 | 利息保障倍数 | < 1 | 1–3 | > 3 |
| 偿债 | 现金流利息保障 | < 1 | 1–2 | > 2 |
| 偿债 | 速动比率 | < 0.5 | 0.5–1.0 | > 1.0 |
| **治理** | 大股东质押比例 | > 70% | 50–70% | < 50% |
| 治理 | 审计意见 | 非标 | 带强调事项段 | 标准 |
| 治理 | 高管/审计机构变动 | 1 年 ≥ 2 次 | 1 年 1 次 | 无 |
| 治理 | 关联交易 / 营收 | > 50% | 30–50% | < 30% |
| 治理 | 监管措施 | 立案/退市预警 | 问询/警示 | 无 |

## D-2：一票否决规则

| 情形 | 判定 | 后续动作 |
|---|---|---|
| 任何一项 🔴 | **不通过** | 估值结果作废，标的剔除 |
| 累计 ≥ 3 项 🟡 | **警示** | 估值结果打 7 折，标记观察 |
| ≤ 2 项 🟡，无 🔴 | **通过** | 进入估值流程 |
| 全部 🟢 | **优质** | 优先纳入候选池 |

> ⚠️ "红灯一票否决"比"加权打分"更稳健——财务造假往往只在一个角落露马脚，平均化会掩盖风险。
>
> 经典反例：康美药业在被立案前，ROE、毛利率、净利率都"很正常"，平均分极高——但货币资金/有息负债的红灯一直亮着。

# E · 行业差异化阈值

**核心原则**：通用阈值仅作初筛，最终判断要回到"同行业历史中位数"和"自身历史均值"。

| 行业 | 需调整的指标 | 调整方向 |
|---|---|---|
| **银行 / 保险** | 资产负债率、ROE、有息负债 | 通用阈值完全不适用，需用拨备覆盖率、不良率、偿付能力充足率等专门指标 |
| **房地产** | 存货周转、合同负债、净负债率 | 存货周转 > 1000 天常态；看销售去化率和"三道红线" |
| **证券** | 资产负债率、杠杆率 | 高杠杆是常态，看净资本和风控指标 |
| **重资产周期股**（钢铁、化工、煤炭） | 利息保障倍数、FCF | 周期底部 FCF 为负正常，看 PB 和景气位置 |
| **科技 / 医药** | 研发资本化 | 容忍度更高，但仍需关注比例突变 |
| **零售连锁** | 应收账款 | 应收占比天然低，重点看存货周转和门店扩张速度 |
| **建筑 / 工程** | 应收账款、合同资产 | 应收占比天然高（工程款回收周期长），看账龄结构 |

# 完整排雷流程

```
1. 拉取数据
   ↓ akshare 三大报表 + 公告类数据
2. 识别行业
   ↓ 申万一级分类
3. 加载阈值
   ↓ 通用阈值 + 行业差异化校准
4. 三大维度扫描
   ↓
   ├─ A. 资产端：商誉/应收/存货/研发/其他应收
   ├─ B. 偿债：杠杆/短期偿债/现金流覆盖
   └─ C. 治理：质押/审计/关联/监管
   ↓
5. 仪表盘汇总（红黄绿）
   ↓
6. 一票否决判定
   ↓
7. 输出排雷报告（含红黄灯的具体证据链）
```

# 关键原则

1. **排雷优先于估值**：再便宜的股票，雷踩了就归零
2. **红灯一票否决**：不要用其他绿灯抵消单一红灯
3. **趋势 > 绝对值**：单期指标好看不算数，看 3–5 年趋势走向
4. **行业差异化**：通用阈值不能套用所有行业，金融/地产/周期需专门规则
5. **附注比正表更重要**：报表造假往往隐藏在附注未披露处
6. **多源交叉**：报表 + 公告 + 审计意见 + 监管函 + 媒体报道，单一来源不可靠
7. **看演变，不看快照**：仪表盘的颜色漂移比当年颜色更重要
8. **业绩好的公司更要排雷**：业绩差的容易识别，业绩"完美"的反而要警惕（康美模式）

---

**一句话总结**：排雷的本质是**"找内部矛盾"**——账上现金多得花不完却还在借款、应收账款比营收涨得快、商誉占了一半净资产、大股东把股票全质押了。只要能用结构化数据发现矛盾，70% 的雷就能在估值前排掉。

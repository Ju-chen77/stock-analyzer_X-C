# CHANGELOG

## [app_V2] — 2026-06-12

### 新增

- **三段式财务分析框架**（对应 `财务分析框架.md`）
  - **Tab ① 业绩检验**
    - KPI 卡片：最新一期营收 / 净利润 / 毛利率 / 净利率 / ROE
    - 营收 & 净利润柱状图 + YoY 增速折线（双轴）
    - 毛利率 / 净利率 / ROE 趋势折线
    - 三组现金流对照图（收入含金量 / 成本含金量 / 利润含金量）
  - **Tab ② 业绩归因**
    - 杜邦三因子分解图（净利率 × 总资产周转 × 权益乘数 = ROE）
    - 杜邦数字明细表（5 年历史）
    - 销售 / 管理 / 研发 / 财务 费用率趋势折线
  - **Tab ③ 验证排雷**
    - 五项排雷仪表盘（红黄绿三色）：利润含金量 / 收入质量 / 库存压力 / 商誉风险 / 偿债风险
    - 应收账款增速 vs 营收增速趋势图
    - 存货增速 vs 营收增速趋势图
  - **Tab ④ 估值区间**（对应 `PE_PB估值法（akshare）.md`）
    - PE 六维矩阵信号卡（TTM PE / 历史分位 / Forward PE / PEG / PB 分位 / EPS 增速）
    - 历史 PE-TTM 折线图（近 5 年 + 中位数线 + 当前值线）
    - 历史 PB 折线图（近 5 年 + 中位数线 + 当前值线）
    - 三角验证目标价区间（低 / 中 / 高）
    - 三档情景分析表（悲观 / 中性 / 乐观）

- 前端改用 **Plotly.js** 交互式图表（CDN 加载），支持悬浮查看数值
- 后端新增 `_get_equity()`：兼容 6 种新浪资产负债表权益列名写法，找不到时用「资产 − 负债」反推
- 后端新增 `_get_pe_history()`：用月线价格 × 年报 EPS/BVPS 本地计算历史 PE/PB 序列
- 后端新增 `compute_valuation()`：PE/PB 矩阵 + 三角验证 + 情景分析完整估值流程

### 修复

- **日期格式 bug**：`_set_date()` 加 `.replace("-", "")` 兼容 `"2016-03-31"` 带横线格式，修复年报过滤全部失败、指标显示 N/A 的问题（由合作者 Claude 检出）
- **权益乘数 / ROE 不显示**：扩展权益列名匹配范围，修复杜邦分解三因子缺失的问题
- **PE/PB 折线图不显示**：显式声明 `xaxis.type:'date'`，去掉 `fill:'tozeroy'` + 透明底色组合（该组合会把线条压到填充层之下）
- **Tab 切换图表贴底**：改为懒渲染，切到对应 Tab 时才调用 Plotly.newPlot()，修复隐藏容器高度为 0 导致所有数据点压在 y=0 的问题
- `fcol()` 改为精确匹配优先，再做子串匹配，减少列名误命中

## [app_V1] — 2026-06-11

### 新增

- Flask 单文件 Web 应用（后端 + 内嵌前端）
- 按股票名称（模糊）或 6 位代码精确搜索
- 自动抓取三大报表关键科目（最近 5 期）
- 三种 PE 口径对比：静态（LYR）/ 滚动（TTM）/ 动态，数据源值 vs 手算值对照
- 总市值展示（现价 × 反推总股本）

### 数据源

- AKShare（免费，无需注册）
- 新浪财务接口：三大报表
- 东财快照：行业、现价
- 雪球快照：PE 三种口径（可选，境外网络可能不通）

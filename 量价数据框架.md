# 量价数据框架

> **核心目的**：用市场行为（成交量、价格、资金流向、异动）**印证或反驳基本面结论**，而不是预测股价。
>
> **定位**：基本面分析的"市场印证层"——**横切**业绩检验 / 业绩归因 / 验证排雷 / 估值四个模块，不构成独立的分析体系。
>
> **核心信号**：基本面 + 量价**共振** = 强信号；基本面 + 量价**背离** = 警示信号。

---

## 一、设计原则

| 原则 | 说明 |
|---|---|
| **印证而非预测** | 量价回答"市场是否同意基本面判断"，不做择时工具 |
| **基本面优先** | 量价信号不能否决基本面结论，但可以增强或警示 |
| **共振信号最强** | 基本面 + 量价同向 = 强信号；背离 = 警示 |
| **异动是触发器** | 异动后回头看基本面找原因，不是因异动直接下结论 |
| **保持简单** | 只用最基础的量价指标，不堆技术分析（MACD/KDJ/布林带） |

---

## 二、四类量价数据

```
┌──────────────────────────────────────────────────────────┐
│                                                           │
│   基础行情          资金流向        异动事件      衍生指标  │
│   ─────────        ─────────       ─────────     ─────────│
│   "怎么走的"        "谁在动"        "发生了什么"  "相对强弱"│
│                                                           │
│   • OHLCV          • 主力净流入     • 涨跌停      • 均线   │
│   • 换手率          • 北向持股       • 龙虎榜      • 量比   │
│   • 振幅           • 融资融券       • 大宗交易     • 相对强度│
│                                                           │
└──────────────────────────────────────────────────────────┘
```

| 数据类别 | 具体指标 | 用途 |
|---|---|---|
| **基础行情** | OHLCV、换手率、振幅 | 趋势、波动 |
| **资金流向** | 主力净流入、北向持股变化、融资融券余额 | 谁在买卖 |
| **异动事件** | 涨跌停、龙虎榜、大宗交易 | 重大资金信号 |
| **衍生指标** | 均线、量比、相对强度（RS） | 趋势 + 强弱 |

# 三、四大应用场景（横切现有模块）

量价数据不构成独立模块，而是**叠加在现有四个基本面模块之上**：

```
                    基本面分析（已有）
   ┌──────────────────────────────────────────┐
   │  业绩检验  →  业绩归因  →  验证排雷       │
   │                                            │
   │             ↓ 估值结论                     │
   │           PE/PB 估值                       │
   └──────────────────────────────────────────┘
                       ↕
   ┌──────────────────────────────────────────┐
   │         量价数据（市场印证层）             │
   │                                            │
   │  A 业绩披露窗口期                          │
   │  B 排雷信号增强                            │
   │  C 估值修复确认                            │
   │  D 资金流向与行业轮动                      │
   └──────────────────────────────────────────┘
```

## 场景 A · 业绩披露窗口期监测

**用途**：检测信息泄露、验证业绩是否被市场认可。

### 三段窗口监测

| 时间窗口 | 监测内容 | 判读 |
|---|---|---|
| **披露前 10 日** | 异常放量 + 涨跌幅 | 信息可能泄露 |
| **披露当日** | 涨跌幅 + 成交量 + 缺口 | 市场对业绩的第一反应 |
| **披露后 5 日** | 是否承接 / 阴跌 | 真实定价 vs 短期博弈 |

### 信号判读矩阵

| 业绩 | 披露前 | 披露当日 | 披露后 5 日 | 解读 |
|---|---|---|---|---|
| 超预期 | 平稳 | 大涨高量 | 持续承接 | 健康，市场认可 |
| 超预期 | 提前大涨 | 高开低走 | 阴跌 | 利好出尽 / 信息泄露 |
| 符合预期 | 平稳 | 微涨 | 平稳 | 正常 |
| 不及预期 | 提前下跌 | 跌停 | 持续阴跌 | 警示 |
| 不及预期 | 平稳 | 跌停 | 反弹 | 利空出尽 |

**整合点**：业绩检验报告输出后，附加"市场反应"小节。

**图表**：✅ 披露日前后 ±15 日的 K 线 + 成交量叠加图

## 场景 B · 排雷信号增强

**用途**：把量价异常作为排雷仪表盘的补充信号。

### 基本面雷 × 量价信号 升级矩阵

| 基本面信号 | 量价同步信号 | 判读升级 |
|---|---|---|
| 大股东质押 > 70% | 放量下跌 + 北向撤出 | 🔴 → 🔴🔴 极高危 |
| 商誉占比 > 50% | 持续阴跌 + 缩量 | 市场已在定价 |
| 审计意见非标 | 跌停 + 巨量 | 即刻退出 |
| 业绩看似优秀 | 大股东减持 + 北向撤出 | "完美业绩"警示 |
| 有息负债 > 货币资金 ×2 | 融资余额激增 + 放量 | 散户接盘机构出货 |
| 关联交易占比高 | 龙虎榜频现游资 | 短期博弈，无价值支撑 |

### 关键警戒模式

| 模式 | 含义 |
|---|---|
| **机构出货 + 散户进场** | 北向 / 大宗交易卖出 + 融资余额上升 |
| **业绩前夜异动** | 披露前 10 日内异常放量下跌 → 暴雷预警 |
| **持续放量阴跌** | 量价齐跌且持续，比缩量阴跌更危险 |

**整合点**：在 `risk_check.py` 的红黄绿仪表盘里**新增"市场信号"行**。

## 场景 C · 估值修复确认

**用途**：低估值不一定立即修复，量价帮你识别"市场什么时候开始动"。

### 估值 × 量价 共振 / 背离矩阵

| 估值状态 | 量价信号 | 判读 | 操作含义 |
|---|---|---|---|
| 历史低位 | 量能温和放大 + 突破均线 | **共振做多** | 进场窗口 |
| 历史低位 | 持续缩量 + 阴跌 | **价值陷阱**，市场不认 | 等待或重审基本面 |
| 历史合理 | 北向持续买入 | 外资认可 | 持有 |
| 历史合理 | 主力净流出 + 阴跌 | 警惕基本面变化 | 复核基本面 |
| 历史高位 | 量价背离（价涨量缩） | 警惕回调 | 减仓信号 |
| 历史高位 | 放量滞涨 | 主力出货嫌疑 | 减仓 |

### 进场信号确认（三重共振）

```
基本面通过        ── PE/PB 历史低位
                     +
量能确认          ── 5日均量较前期放大 30%+
                     +
趋势确认          ── 价格站上 60 日均线 + 均线多头排列
                     ↓
              强信号：估值修复启动
```

**整合点**：PE/PB 估值结论后，附加"市场是否启动"判断。

## 场景 D · 资金流向与行业轮动

**用途**：判断个股 alpha 还是行业 beta，识别行业景气切换。

### 相对强度（Relative Strength）

| 指标 | 计算 | 判读 |
|---|---|---|
| **个股 RS** | 个股涨幅 − 行业指数涨幅 | 行业内排序 |
| **行业 RS** | 行业指数涨幅 − 大盘涨幅 | 行业景气位置 |
| **个股 RS 排名** | 个股 RS 在行业内的百分位 | > 80% 即"龙头股" |

### 资金流向监测

| 指标 | 周期 | 用途 |
|---|---|---|
| 主力净流入 | 5 日 / 20 日累计 | 短期资金共识 |
| 北向资金持股 | 月度变化 | 外资观点（长期 alpha 信号） |
| 融资余额 | 周变化 | 散户/中小机构杠杆情绪 |
| 大宗交易 | 单笔 + 累计 | 大资金主动行为 |

### 行业轮动信号

| 信号 | 判读 |
|---|---|
| 行业 RS 从历史低位反转 + 板块放量 | 行业景气切换早期 |
| 行业 RS 高位 + 个股 RS 走弱 | 板块见顶迹象 |
| 行业内龙头率先放量 | 行业上行确认 |
| 行业指数破位 + 龙头跌停 | 板块下行信号 |

**整合点**：独立的"市场情绪"模块，给基本面分析提供宏观背景。

# 四、信号汇总：基本面 × 量价 四象限

```
                  量价正向（市场认可）
                         ↑
                         │
      共振警示           │           共振做多
   （高估值 + 放量）      │      （低估值 + 放量+突破）
                         │
基本面差 ←───────────────┼───────────────→ 基本面好
                         │
                         │
      静默回避           │           价值陷阱
   （差 + 缩量阴跌）      │      （好 + 缩量阴跌）
                         │
                         ↓
                  量价负向（市场冷淡）
```

| 象限 | 基本面 | 量价 | 操作含义 |
|---|---|---|---|
| **右上** | 好 | 正向 | **共振做多**，最强信号 |
| **左上** | 差 | 正向 | **共振警示**，泡沫/炒作 |
| **右下** | 好 | 负向 | **价值陷阱**，待确认 |
| **左下** | 差 | 负向 | **静默回避**，避开即可 |

# 八、关键陷阱与规避

| 陷阱 | 应对 |
|---|---|
| **过度依赖技术指标** | 只用最基础的均线、量能、换手率，不堆 MACD/KDJ/布林带 |
| **量价信号高频化** | 用日 K 不用分钟 K，避免高频噪音 |
| **回测过拟合** | 不为量价规则做参数优化，只做共振/背离的定性判断 |
| **本末倒置** | 基本面必须先得出结论，再用量价验证——不能反过来 |
| **追逐异动** | 涨跌停、龙虎榜只作为"回头看基本面"的触发器，不直接下结论 |
| **忽视市场情绪周期** | 牛市 / 熊市 / 震荡市下，同样的量价信号含义不同 |

---

# 九、关键原则

1. **量价不预测，只印证**：永远先有基本面结论，再用量价检验
2. **共振优先于单边**：基本面好 + 量价正向 = 才是真信号
3. **背离比一致更值得关注**：背离意味着"有一方错了"，需要深挖
4. **简单胜过复杂**：基础指标（量、价、换手、北向、主力）已足够
5. **趋势 > 当日**：单日异动可能是噪音，5–20 日趋势才有意义
6. **机构信号 > 散户信号**：北向、大宗交易比融资余额更可靠
7. **行业 beta 与个股 alpha 分开看**：先判断行业景气，再看个股相对强度

---

**一句话总结**：量价数据不是新增一个独立的分析体系，而是给现有的基本面分析加一层"市场印证"——**基本面告诉你"该不该买"，量价告诉你"市场是否同意、资金是否已在动"**。两者共振才是真信号；两者背离则要么基本面错了，要么市场错了——值得深挖。
